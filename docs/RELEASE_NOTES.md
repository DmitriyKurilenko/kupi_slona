# Release Notes

## 2026-05-17

### Новое
- MVP полностью работоспособен: покупка слонов с уникальными цветами, генерация изображений, подарочные ссылки, интеграция платежей
- Два тарифа: Basic (случайный цвет) и Advanced (выбор оттенка или точного цвета)
- Система подарков с публичными UUID-ссылками
- OAuth-авторизация через Google (django-allauth)
- REST API с автодокументацией (Django Ninja) на `/api/docs`
- Публичный поиск слонов по цвету или имени (`/api/elephants/lookup/`)

### Улучшения
- Интерфейс на русском языке с адаптивной вёрсткой (Tailwind CSS + Alpine.js)
- SEO-оптимизация (meta description)
- Самостоятельный хостинг шрифтов (без Google Fonts)
- Оптимизация производительности: собранный Tailwind CSS, select_related в запросах, Redis-кэширование
- Поддержка HUE-формата для выбора оттенка в Advanced тарифе
- Email-уведомления (настроен SMTP через Beget)

### Исправления
- Повышен rate limit для Google OAuth (с 5r/m до 5r/s) для устранения ошибки 429
- Исправлен health check: теперь используется nginx (443) вместо прямого порта контейнера
- Исправлен deploy-скрипт: сброс nginx-конфигурации перед pull, пере-применение домена из .env
- Переход на email-only аутентификацию (убрано требование username), исправлена обработка пустого last_name в OAuth
- Удалены устаревшие индексы из elephants и payments (миграции)
